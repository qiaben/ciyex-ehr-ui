# Ciyex EHR - Electronic Health Records UI

Open source EHR frontend built with Next.js, React, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js 15** with App Router
- **React 19** with TypeScript
- **Tailwind CSS** for styling
- **Keycloak** for authentication (OAuth2 + PKCE)
- **FHIR R4** for health data interoperability
